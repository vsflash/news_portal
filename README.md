News Portal
-----------

This is demo projects on Symfony 4 framework.

Installation
------------

1. Clone repository

    ```bash
    $ git clone https://github.com/greeflas/ITEA-PHP-Adv6_News-Portal news_portal
    $ cd news_portal
    ```
    
2. Install dependencies

    ```bash
    $ composer install
    ```
    
3. Run server

    ```bash
    $ symfony serve
    ```
